## 登录与账号

- 首次使用点右上角**注册**：输入邮箱 + 密码（≥6 位），系统发送确认邮件，**点击邮件里的链接**完成验证后即可登录。
- 之后用邮箱 + 密码登录；右上角可退出。
- 账号由 Supabase 托管，密码加密存储。忘记密码可重新注册或联系管理员。

## 通用操作（单位 / 主题）

- **单位制切换**：导航栏有「输入 / 输出」两个独立开关，可分别选 **公制（SI）** 或 **美制（IP）**。
  - 切「输入制」→ 输入框数值按新单位自动换算；切「输出制」→ 结果、表头、图表坐标即时重绘。
  - 选择会被记住（下次打开保持）。
  - 温度 °C↔°F、压力 kPa/MPa↔psi、流量 m³/h↔GPM/CFM 等全站生效。
- **主题**：导航栏切换 深色 / 浅色 / 高对比，三种随喜好。仿真页场景固定深色。
- **服务状态**：首页顶部显示后端与 CoolProp 是否在线。

## 2. 系统仿真

**用途**：变频一拖一热泵的动态可视化仿真，直观理解制冷/制热/化霜/回油四种工况下冷媒循环与部件动作。**定位：教学定性模型**（趋势正确，非精确设计工具）。

**四种工况**（顶部按钮）：制冷 / 制热 / 化霜 / 回油。

**9 个可调参数**（左侧滑块）：
| 参数 | 说明 |
|---|---|
| 压缩机转速 | 20–120 Hz |
| 内风机 / 外风机 | 0–100 % |
| EXV 开度 | 5–100 %（同时显示脉冲数 P） |
| 冷媒量 | 40–130 %（模拟缺氟/过充） |
| 内 / 外换热器 | 50–150 %（模拟脏堵/结垢） |
| 室内 / 室外温度 | 工况环境 |

**观察区**：
- **场景动画**：冷媒粒子按相态（气/液/两相）以不同形状+颜色流动，流速∝流量；压缩机涡旋、四通阀换向、风机、EXV 弧圈、结霜等动画。
- **压焓图**：实时绘制 1-2-3-4 循环与饱和穹顶。
- **测点仪表**：排气/回气压力、过热度、过冷度、各段温度，以及**节流部件与蒸发器进出口状态**（节流前过冷液、节流后两相+干度、蒸发器进/出口温压）。数值变化时自动高亮。
- **性能指标**：制冷/制热量、输入功率、COP、质量流量、压比。
- **诊断提示**：过热度过低/过高、过冷异常、排温过高等实时提示。

**操作**：选工况 → 拖动滑块改变参数 → 观察循环、测点与性能如何响应。顶部可切公制/美制。

## 3. 冷媒物性

**用途**：查询 12 种冷媒任意状态点的热力学与输运物性（CoolProp 高精度）。

**冷媒**：R410A · R32 · R454B · R134a · R290 · R404A · R407C · R22 · R1234yf · R600a · R717(氨) · R744(CO₂)。

**输入**：
1. 选冷媒；
2. 选**已知参数组合**（TP 温压 / PQ 压力+干度 / TQ 温度+干度 / PH 压力+焓 / PS 压力+熵）；
3. 填对应两个参数值（标签随组合自动变化）。

**输出**：焓 / 熵 / 密度 / 干度 / 比热 / 黏度 / 导热系数 + 相态与过热/过冷度判定；可生成**饱和表**。混合冷媒（R454B/R407C 等）报告泡点/露点与温度滑移。

**注意**：两相区 T 与 P 不独立，用 TP 组合会提示改用 TQ/PQ。焓熵采用 **IIR 基准**（0°C 饱和液 h=200 kJ/kg），与常用物性表一致。

## 4. 压焓计算

**用途**：理论单级压缩循环计算 + 真实压焓图。

**输入**：
- 冷媒；
- **输入方式**：蒸发/冷凝**压力** 或 **温度**；
- 对应蒸发、冷凝值；
- **压缩等熵效率**（默认 0.70，可改 0.3–1）。

**输出**：焓差、COP、排气温度、单位容积制冷量、制冷/热量；压焓图上绘出循环。

**交互**：压焓图上 4 个状态点**可拖拽**——点 1/4 上下改蒸发压力、点 2/3 上下改冷凝压力、点 1 左右改过热度、点 3 左右改过冷度，松手自动 CoolProp 精算；鼠标悬停读 (h, P, 饱和温度)。

## 5. 湿空气

**用途**：湿空气状态计算（ASHRAE RP-1485 / CoolProp HAProps）+ 焓湿图。

**输入**：选**已知参数组合**（干球、湿球、相对湿度、露点、含湿量、焓 中的任意两个），填两个值；可设海拔/大气压。

**输出**：其余全部湿空气参数；状态点落在**焓湿图**上，带引导线。图上含 RH 10–100% 曲线族与等焓线，**可点击图面取点**（干球+含湿量）自动精算。

**附加**：**水蒸气饱和换算**——温度 ↔ 饱和压力互算。

## 6. 水力计算

**用途**：闭式/开式水系统分段管路阻力 → 泵扬程/轴功率 → 输送能效 EC(H)R 判定。

**输入**：
- **输送介质**：水 / 30% 乙二醇 / 50% 乙二醇（CoolProp 物性）；
- 各**管段**：流量、管材（钢/PPR/铜，内径库）、管径、长度、管件当量（ΣK）；
- **扬程安全系数**（默认 1.10）、**水泵效率 η**（默认 0.65）；
- **EC(H)R 限值系数** A / B / α（默认 GB 50189 量级，可按分档改）。

**输出**：各段沿程+局部阻力（Swamee-Jain 摩阻）、总扬程、轴功率、EC(H)R 实际值与限值对比判定；流速超限标红。

**注意**：EC(H)R 的 A/B/α 请按 GB 50189-2015 对应冷热水/规模分档核对后再用。

## 7. 风管设计

**用途**：等摩阻法风管选型 + 系统静压校核。

**输入**：各**管段**风量、目标单位摩阻、管型（圆/矩形），风机机外静压（ESP）等；填好后点「**校核并生成示意图**」。

**输出**：反求管径 + 推荐标准圆管/矩形组合（当量直径、长宽比≤4），分段静压累加，机外静压校核（裕量≥10% 判定）；自动生成**最不利风路示意图**（风机→各段→末端，宽∝长度、高∝管径、超速段红色告警，逐段标注尺寸/风量/流速/ΔP）。

## 8. 能耗电费

**用途**：北美 SEER2/HSPF2 机组全年耗电与电费估算、双机型对比。

**输入**：
- **州**（美国，选后自动带出该州参考 CLH/HLH/电价，可改）；
- **制冷满负荷小时 CLH**、**制热满负荷小时 HLH**、**电价（¢/kWh）**；
- **机型 A**：SEER2 / HSPF2；**机型 B**（选填，用于对比）。

**输出**：当量满负荷小时法算出的全年制冷/制热 kWh 与电费；A/B 双机型对比柱图。

**注意**：内置各州 CLH/HLH/电价为 ENERGY STAR / EIA 量级参考值，**请按当地实际修改**；极寒地区电辅热未计入。

## 9. 单位换算

**用途**：暖通常用 12 类单位互换。输入数值、选源单位，**即时换算**出各目标单位。

**类别**：长度、面积、体积、质量、温度、压力、体积流量、流速、功率/冷量、能量、密度、传热系数。

## 常见问题

**Q：打开提示要登录？**
A：站点默认开启登录门禁。注册并通过邮件验证后即可使用。

**Q：计算页点了没反应 / 报错？**
A：物性计算依赖后端 CoolProp。首页看服务状态；网络波动时稍后重试。仿真页为纯前端，不受影响。

**Q：数值单位不对？**
A：检查导航栏「输入 / 输出」单位制开关（公制/美制），两者独立。

**Q：升级后显示还是旧的？**
A：强制刷新浏览器缓存（Ctrl+F5）。站点资源带版本号，一般会自动更新。

**Q：仿真页的数能当设计依据吗？**
A：`sim` 是教学定性模型，看趋势用；精确计算请用 `refprops` / `phcalc` / `psychro`（CoolProp 高精度）。
